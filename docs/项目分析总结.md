## 项目定位

这是一个 **Claude Code 最佳实践大全**，口号是 "practice makes claude perfect"。

它**不是**一个可运行的应用程序，而是一个**知识型资源仓库**，系统性地整理了如何高效使用 Claude Code 的方法论、工作流、技巧和实战经验。

- 覆盖了 Claude Code 的**所有**核心功能模块
- 每个模块都有前置字段说明和官方内置功能列表
- 从概念到实践的完整路径

### 2. 实战导向
- 不只是文档翻译，整合了核心团队成员的推文和经验
- 提供了完整的编排工作流示例
- 35+ 条分类整理的实用建议

### 3. 持续更新
- 紧跟 Claude Code 版本（当前 v2.1.76）
- 每次更新都有 changelog 追踪
- 报告中包含版本间对比分析
